About Me

BSc Honours in Computing & Cyber Security

At around 2003-04 when I was 7-8 years old, I received my first computer that brought me into the whole computing world. I started as everyone else learning the basic’s about computers. As time passed, my passion for computers started to grow which brought my interest to Programming, I wanted to understand how computers actually work and how to make them do what I wanted.

When I was around 13-14 years old, I started to plan studding computer Science in University so in order to get a head start, I started to make some basic HTML & CSS websites. As I went to University, I started to learn new programming languages and I started to notice how important security is in our daily tasks. My passion for Programming has moved to Cyber Security as this area got me very interested and created a lot of challenges to overcome.

In my free time I started to learn about security and how to perform various of manual attacks as also on how to stop them. I have worked with tools that were using for analytical purpose, tracking programs as also programs performing exploits like commit, SQLMAP, BurpSuite or even Metasploit that gave me understanding of how easy can it be to get into a software / system / application that is not secured properly.

My plans for future are to work within Cyber Security company that would allow me to learn and grow within the company being able to give 200% from myself.
